### Run Locally

Clone the project

```bash
  git clone https://github.com/fznoor/react-ecom.git
```

Go to the project directory

```bash
  cd react-movie
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
